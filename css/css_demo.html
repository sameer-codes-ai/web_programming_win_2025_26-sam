Cascading Style Sheets (CSS)

CSS is used to style and format HTML elements. HTML defines structure, CSS defines appearance (color, size, layout, spacing). Styles are applied based on priority rules when multiple CSS rules affect the same HTML element. CSS follows an order of precedence to decide which style wins.
